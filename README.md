# CodingFirst (coding_first_web_cli_new)

A Quasar Framework app

## 安装依赖
```bash
npm install
```

### 以开发模式启动APP (热部署，错误报告等)
```bash
quasar dev
```

### 格式化代码
```bash
npm run lint
```

### 以生产模式构建APP
```bash
quasar build
```

### 自定义设置
See [Configuring quasar.conf.js](https://quasar.dev/quasar-cli/quasar-conf-js).
