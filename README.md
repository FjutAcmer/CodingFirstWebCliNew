# CodingFirst (coding_first_web_cli_new)

A Quasar Framework app

## 安装依赖

```bash
npm install
```

### 以开发模式启动 APP (热部署，错误报告等)

```bash
quasar dev
```

### 格式化代码

```bash
npm run lint
```

### 以生产模式构建 APP

```bash
quasar build
```

### 推送到 Github 个人主页

在 Github 中添加一个发布仓库，命名为 github-publish，然后运行命令

```bash
npm run deploy
```

### 自定义设置

戳 [Configuring quasar.conf.js](https://quasar.dev/quasar-cli/quasar-conf-js).
